Описание библиотек используемых в проекте:

torch
Основная библиотека для глубокого обучения от Facebook AI Research.
Для чего: Создание и обучение нейронных сетей
Особенности: Динамические графы вычислений, GPU-ускорение, автоматическое дифференцирование

torchvision
Компаньон для torch, специализируется на компьютерном зрении.
Для чего: Датасеты, модели, трансформации для изображений
Содержит: Предобученные модели (ResNet, VGG и др.), утилиты для загрузки данных

torchaudio
Аналог torchvision для аудио.
Для чего: Обработка аудиоданных, аудиомодели
Содержит: Аудиодатасеты, преобразования сигналов


Работа с изображениями и видео
pillow / PIL (Python Imaging Library)
Pillow - форк PIL, базовая библиотека для работы с изображениями.
Для чего: Открытие, сохранение, манипуляция изображениями
Форматы: JPEG, PNG, BMP, GIF и другие

opencv-python (OpenCV)
Библиотека компьютерного зрения реального времени.
Для чего: Обработка изображений/видео, детекция объектов, фильтры
Особенности: Высокая производительность, множество алгоритмов
Примеры: чтение видео, распознавание лиц, трекинг объектов


Модели и эмбеддинги
open-clip-torch==2.0.2
Реализация CLIP (Contrastive Language-Image Pretraining) от OpenAI.
Для чего: Связывание текста и изображений в одном векторном пространстве
Как работает: Обучает модель понимать соответствие между текстом и изображениями
Применение: Поиск изображений по тексту, классификация, генерация

easyocr
Готовый OCR (Optical Character Recognition) движок.
Для чего: Распознавание текста на изображениях
Особенности: Поддержка множества языков, предобученные модели
Простота использования: Всего несколько строк кода для базового распознавания


Работа с векторными данными
faiss-cpu (Facebook AI Similarity Search)
Библиотека для эффективного поиска похожих векторов.
Для чего: Поиск ближайших соседей в векторных пространствах
Применение: Поиск похожих изображений/текстов, рекомендательные системы
Особенности: Оптимизирована для больших наборов данных


Графический интерфейс
pyqt6
Фреймворк для создания десктопных приложений с GUI.
Для чего: Создание кроссплатформенных приложений
Особенности: Множество виджетов, поддержка разных ОС
Альтернативы: Tkinter (встроен в Python), PySide


Утилиты
tqdm
Индикатор прогресса для циклов и итераций.
Для чего: Визуализация прогресса выполнения длительных операций
Простота: Обернуть любой итератор в tqdm()
Польза: Удобно для обучения моделей, обработки больших данных


Библиотеки для нечёткого поиска
thefuzz>=0.19.0
python-Levenshtein>=0.12.0  # опционально для скорости
Основное назначение
Эти библиотеки используются для нахождения похожих строк, даже если они содержат опечатки, разные регистры, лишние пробелы или небольшие отличия.
TheFuzz (ранее fuzzywuzzy)
Что делает?
Вычисляет степень схожести между строками по алгоритму Левенштейна (редакционное расстояние).

Python-Levenshtein
Что делает?
Реализует алгоритм Левенштейна — вычисляет минимальное количество операций (вставка, удаление, замена символа), чтобы превратить одну строку в другую.
Зачем нужна отдельно от TheFuzz?
Скорость: C-реализация в 4-10 раз быстрее pure Python
Точность: Низкоуровневые функции для специализированных задач

Где это применяется на практике?
E-commerce: Поиск товаров с опечатками
CRM системы: Объединение дубликатов контактов
Биоинформатика: Сравнение последовательностей ДНК
Обработка форм: Корректное понимание пользовательского ввода
Анализ логов: Группировка похожих сообщений об ошибках
Машинное обучение: Препроцессинг текстовых данных
Парсинг данных: Сопоставление данных из разных источников
